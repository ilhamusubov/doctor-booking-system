# Doctor Booking System

Backend REST API for managing doctor appointments.

## Tech Stack

* Java 17
* Spring Boot
* Spring Security
* PostgreSQL
* Redis
* Docker

## Features

* JWT authentication
* Role-based access (ADMIN, DOCTOR, PATIENT)
* Appointment booking
* Review system
* Validation and global exception handling

* ## About

This project is built as a backend system for managing doctor-patient interactions. 
It allows patients to book appointments, leave reviews, and interact with doctors, 
while administrators can manage users and approvals. The system also includes 
secure authentication and role-based access control.
